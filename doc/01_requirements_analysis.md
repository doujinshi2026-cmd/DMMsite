# 要件分析

# プロジェクト目的

FANZA同人作品を紹介するSEOサイトおよびX自動投稿BOTを構築し、

- アフィリエイト収益化
- SEO流入獲得
- SNS拡散
- 技術習得

を実現する。

---

# 背景

従来のFANZA BOTは、

- API転載
- ランキング貼り付け
- リンク集

になりやすく、

- DMM審査落ち
- X凍結
- CTR低下
- SEO低評価

が発生しやすい。

また、単純な作品紹介だけではユーザーは購入しない。

本システムでは、

「感情導線」

を重視する。

---

# ユーザー心理分析

FANZA同人購入は、

「検索購入」

より、

「感情購入」

が強い。

ユーザーは：

- 性癖
- 空気感
- キャラ感情
- 関係性
- 背徳感
- 続きが気になる感覚

によって購入行動を行う。

---

# SNSで強い導線

現在Xで伸びている投稿は、

- 試し読み
- スレッド形式
- 1/3,2/3,3/3構成
- 続きが気になる構成

を採用している。

これは：

「無料サンプル最適化」

である。

---

# 解決方針

以下によって解決する。

- DMM APIによる自動取得
- 試し読み画像抽出
- 感情フック生成
- スレッド投稿
- Markdown記事生成
- SEOレビュー記事
- Next.js静的生成
- GitHub Actions自動化
- X BOT投稿
- SQLite履歴管理

---

「作品紹介BOT」

ではなく、

「続きを読みたくなる感情導線」

を作る。
